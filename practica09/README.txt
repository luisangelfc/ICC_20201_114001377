
>> Para poder ejecutar el programa
	En ese programa no vi muy necesario usar gradle entonces lo que tenemos que hacer es ubicarnos en la carpeta de /practica09 y escribir los siguientes comandos

	javac Writter.java
	java Writter

Y con esto se generará un archivo calculos.txt (dentro de /practica09) en el cual estarán hechos los calculos pertinentes. :)
