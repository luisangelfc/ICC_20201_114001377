- Para poder compilar dirijase a la carpeta src.
practica04/src$

Después ejecute el comando:
javac main/Main.java

Esto para que se compile todo lo necesario.

- Después para ejecutar, en la misma carpeta ingrese el comando:
java main/Main.java > Figuras.html

Y para ingresar a la pagina tiene que entrar a la carpeta a tráves de "Archivos", ingresar a la carpeta practica04/src y abrir el html llamado Figuras (le puede poner el nombre que desea desde la terminal).
